Навигация по сайту

1. Лента новостей находится слева. В ней указаны заголовок, источник и время. Для просмотра выберите новость в списке.

2. Карточка новости находится справа. В карточке размещены краткое содержание и основные сведения по событию.

3. Оригинальная публикация доступна по ссылке в карточке. Переход по ссылке открывает первоисточник.

4. Аналитический блок расположен в карточке и содержит автоматический вывод по событию: оценку влияния (позитив/негатив/нейтрально) и комментарий

Источники

ADNews агрегирует новости и официальные сообщения из открытых источников. В числе используемых источников: ТАСС, РБК, Ведомости, Investing.com, а также публикации регуляторов и ведомств (например, ФРС, ЕЦБ, SEC) и другие профильные деловые медиа.

О проекте

ADNews — информационно-аналитический сервис мониторинга новостей финансовых рынков. Платформа агрегирует публикации из надежных источников, выполняет их тематическую классификацию и формирует структурированные краткие выводы по событиям.

Проект выполнен при поддержке «Фонда содействия инновациям» в рамках программы "Студенческий стартап" федерального проекта «Платформа университетского технологического предпринимательства»."
